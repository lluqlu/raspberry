# 🍓 Raspberry Bridge Node

> Nodo de intercambio de archivos entre la Raspberry Pi y otros dispositivos de la red.

Un proyecto pensado para convertir una Raspberry Pi 5 en un **punto central de transferencia de archivos**, permitiendo compartir documentos entre PC, celulares, notebooks u otros equipos de forma simple, rápida y autónoma.

---

## 📌 Objetivo

Usar la Raspberry como un **puente inteligente** dentro de la red local o remota para:

- 📂 Compartir archivos entre dispositivos
- 📥 Recibir documentos desde celulares
- 📤 Enviar archivos a otras PCs
- 🖨️ Centralizar documentos para impresión
- ☁️ Mantener una carpeta común accesible
- 🌐 Acceso remoto seguro (opcional)

---

## 🧠 Concepto

La Raspberry funciona como un **nodo intermedio**, siempre encendido y disponible.

Ejemplo:

```text
Celular → Raspberry → PC
Notebook → Raspberry → Impresora
PC Oficina → Raspberry → Tablet
